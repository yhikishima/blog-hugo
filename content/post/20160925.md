# pumaとは
concurrentなWebサーバー。ワーカー毎にスレッドを立ち上げる。
production環境ではサイトのパフォーマンスを上げるため、複数のワーカーを立ち上げるが、pumaはスレッドで動作するため、排他処理を行える。


## インストール
Gemfileに
```
gem 'puma'
```
と記述し、

```
bundle install
```
でインストールできる。

## 起動
```
rails s Puma
```
でpumaのサーバー（3000番）が立ち上がる。
